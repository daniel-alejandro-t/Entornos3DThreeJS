# Entornos 3D con Three.js

## Instalación de NPM y Node.js en Windows

link de descarga: https://nodejs.org/es/download/

Comprobar que se ha instalado correctamente:

´´´
node --version

npm --version
´´´

## Ejecución del proyecto

´´´
npm run dev
´´´
